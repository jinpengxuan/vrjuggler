# System Configurations for Using VR Juggler #

On this page, we have links and information about what needs to be installed on a '''new''' system in order to use VR Juggler 1.1 and beyond.  This is based on personal experience and on the basic dependencies of VR Juggler.  We focus on VR Juggler 1.1 and newer because that is the code base that is under active development.
